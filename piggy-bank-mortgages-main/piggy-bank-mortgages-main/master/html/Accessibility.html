Accessibility

Accessibility

This website is developed and maintained by Piggy Bank. We want as many people as possible to be able to use this website. This means that you should be able to:

  • Zoom in up to 200% without the text splitting off the screen
  • Navigate most of the website using just a keyboard
  • Navigate most of the website using speech recognition software
  • Clearly understand the content and insrtuctions on the website

AbilityNet has advice on making your device easier to use if you have a disability.

How accessible is this website
  • Compliant with the Web Content Accessibility Guidelines version 2.0 to at least AA standard.
What to do if you can not use parts of the website

If you need further infromation or assistance with any part of the website refer to Help/FAQs or you can also Contact us.

Reporting accessibility problems with this website

We are always looking for ways we can make this website more accessible. If you find problems that are not listed on this page you can send us an email here. Or you can also Contact us.
